## Fast Track (3 Steps):

### 1. Deploy Backend
- Go to https://dashboard.render.com
- Click **New +** → **Web Service**
- Choose repo: `elmiseswatini`
- Environment: **Docker**
- Dockerfile path: `eswatini_lmis_backend/Dockerfile`
- Click **Advanced** → Add environment variables below
- Click **Create Web Service**

### 2. Set Environment Variables
Copy and paste in Render:
```

DB_USER=eswatini_lmis_user
DB_NAME=eswatini_lmis

JWT_SECRET=eswatini_lmis_jwt_secret_2026_secure_key

```

### 3. After Deployment (5-10 min)
You'll get a URL like: `https://lmis-backend-xxxxx.onrender.com`
