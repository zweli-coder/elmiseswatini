🚀 RAILWAY.APP - EXACT COMMANDS FOR CMD
========================================

Copy and paste these commands EXACTLY into Command Prompt!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: NAVIGATE TO YOUR PROJECT FOLDER
═════════════════════════════════════════

Copy and paste this:

cd C:\Users\Sukati\my-app

Press ENTER

(You should see: C:\Users\Sukati\my-app>)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 2: INITIALIZE GIT REPOSITORY
═════════════════════════════════

Copy and paste this:

git init

Press ENTER

(You should see: Initialized empty Git repository)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 3: ADD ALL FILES
═════════════════════

Copy and paste this:

git add .

Press ENTER

(No output, that's normal)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 4: COMMIT FILES
════════════════════

Copy and paste this:

git commit -m "LMIS Application"

Press ENTER

(You should see: create mode... messages)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 5: RENAME BRANCH TO MAIN
══════════════════════════════

Copy and paste this:

git branch -M main

Press ENTER

(No output, that's normal)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 6: ADD REMOTE REPOSITORY
══════════════════════════════

⚠️ REPLACE: nebula20 with YOUR GitHub username

Copy and paste this (with YOUR username):

git remote add origin https://github.com/nebula20/lmis-app.git

Press ENTER

(No output, that's normal)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 7: PUSH TO GITHUB
══════════════════════

Copy and paste this:

git push -u origin main

Press ENTER

(It will ask for GitHub credentials - enter them)

Wait for completion...

(You should see: everything up-to-date or branch created)

✅ Your code is on GitHub!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

NOW GO TO BROWSER FOR RAILWAY.APP SETUP
═════════════════════════════════════════

1. Open browser
2. Go to: https://railway.app
3. Click "Start Project"
4. Sign up with GitHub
5. Authorize Railway
6. Select your lmis-app repository
7. Click Deploy

Wait for deployment... (5-10 minutes)

You'll get permanent URLs!

✅ Your app is LIVE!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

TROUBLESHOOTING
═══════════════

Problem: "git: command not found"
Solution: Install Git: https://git-scm.com/download/win

Problem: "fatal: not a git repository"
Solution: Make sure you ran "git init" first

Problem: "fatal: Permission denied"
Solution: Check your GitHub credentials

Problem: "branch already exists"
Solution: Your repo exists. Skip to Railway setup.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
