## 🎬 Animation Details

### Card Hover Animation (0.4s)
```
Duration:   0.4 seconds
Easing:     cubic-bezier(0.34, 1.56, 0.64, 1) [bouncy effect]
Effects:
  - Card lifts: translateY(-8px)
  - Card scales: scale(1.02)
  - Shadow grows: 0 4px 12px → 0 12px 28px
  - Border appears: transparent → #3498db
  - Background changes: white → light blue gradient
  - Top border slides in: scaleX(0) → scaleX(1)
```

### Button Hover Animation (0.3s)
```
Duration:   0.3 seconds
Easing:     cubic-bezier(0.34, 1.56, 0.64, 1) [bouncy effect]
Effects:
  - Button lifts: translateY(-2px)
  - Shadow grows: 0 4px 12px → 0 6px 20px
  - Gradient reverses (smooth color shift)
```

### Text Color Changes
```
Duration:   0.3 seconds
Effects on hover:
  - Title: #2c3e50 → #3498db (blue)
  - Description: #7f8c8d → #555555 (darker)
  - Meta info: #95a5a6 → #7f8c8d (brighter)
  - Category badge: scales to 1.05
```
