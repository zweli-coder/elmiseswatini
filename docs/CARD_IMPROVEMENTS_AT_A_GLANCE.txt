## 🔢 Animation Numbers

### Timing
```
Card Hover Duration:    0.4 seconds (400ms)
Button Hover Duration:  0.3 seconds (300ms)
Text Color Change:      0.3 seconds
Text Shift Amount:      2-8px
Scale Factor:           1.0 → 1.02
Easing Function:        cubic-bezier(0.34, 1.56, 0.64, 1)
```

### Effects Happening Simultaneously on Card Hover
1. **Top border** slides in from left (scaleX: 0 → 1)
2. **Card lifts** up by 8px (translateY: 0 → -8px)
3. **Card scales** larger (scale: 1.0 → 1.02)
4. **Shadow grows** and changes color
5. **Background** changes to light blue gradient
6. **Border appears** (transparent → #3498db)
7. **Title color** changes to blue
8. **Description darkens**
9. **Meta info slides right** (2px)
10. **Category badge enlarges** (1.05x)
11. **Footer background** changes

**All in 0.4 seconds** with smooth bouncy easing!

## 🎓 Technical Implementation

### CSS Used
```
- linear-gradient() for gradients
- cubic-bezier() for easing
- transform: translateY(), scale(), scaleX()
- transition: for smooth animations
- box-shadow: for depth effects
- ::before pseudo-element for top border
```

### React Used
```
- useState for viewingPublication state
- Conditional rendering for modal
- Event handlers for interactions
- Icon components (FaEye)
- Event.stopPropagation() for modal
```
